# csglobal_CSC500-1-milestone-1

Description

This Python program allows users to enter items they want to purchase, including the item name, price, and quantity. It calculates and displays the total cost for each item and the overall total for all items entered.

How It Works

The user enters how many items they want to purchase.

For each item, the program asks for:

Item name

Item price

Item quantity

The program prints the cost of each item and the total cost at the end.

Class Details

Class: ItemToPurchase

item_name: Name of the item

item_price: Price of the item

item_quantity: Quantity of the item

print_item_cost(): Prints the total cost for that item
